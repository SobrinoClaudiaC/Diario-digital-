Le damos la bienvenida a Diario Digital!

En este mini-proyecto se realizará paso a paso como aplicar el Backend para registrar y autenticar a usuarios almacenados en una base de datos con apoyo de la herramienta Firebase que nos dará múltiples facilidades para que nuestra página funcione correctamente.
Asímismo, realizaremos un CRUD de tareas que permitirá al usuario interactuar con la página creando posts, los cuáles podrán editar y eliminar una vez que se haya registrado o iniciado sesión con el correo de su preferencia, permitiendo también una fácil accesibilidad con servicios como Google directamente.
